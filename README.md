## 📌 Contexte

Dans le cadre de la formation des alternants, les parcours pédagogiques sont souvent **peu engageants** et **peu personnalisés**. Les formateurs manquent d'outils simples pour créer des contenus interactifs adaptés à leur public.

**Edunova** répond à ce besoin en offrant :
- Une interface intuitive pour que les **formateurs créent des modules interactifs** sans compétences techniques
- Un espace personnalisé pour que les **alternants progressent à leur rythme**
- Un tableau de bord de suivi des apprentissages en temps réel
